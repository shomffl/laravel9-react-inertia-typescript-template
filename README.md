## Usage

```
$ git clone git@github.com:shomffl/laravel9-react-inertia-typescript-template.git

$ cd laravel9-react-inertia-typescript-template

$ composer install && npm install

$ cp .env.example

$ ./vendor/bin/sail up -d

$ ./vendor/bin/sail artisan key:generate

$ ./vendor/bin/sail npm run dev

```
